## Projet de Gestion des Réservations de Spectacles
# Présentation du Projet
Ce projet vise à moderniser et à informatiser la gestion des réservations de spectacles pour une société de production. Notre système offre une solution complète pour gérer un catalogue détaillé de spectacles, incluant les auteurs, les metteurs en scène, les comédiens, ainsi que les lieux et les dates de représentations.
Les visiteurs du site peuvent consulter librement le catalogue des spectacles, visualiser les lieux et les prochaines dates de représentation. Ils ont également la possibilité d'effectuer des recherches, des tris et des filtres à travers les différentes pages du catalogue.
Les membres enregistrés bénéficient de fonctionnalités supplémentaires, telles que la réservation de places pour les représentations, la consultation de leurs réservations et la modification de leurs informations de profil.
Les gestionnaires du système ont accès à des outils avancés pour gérer les spectacles, les auteurs, les metteurs en scène, les comédiens, les lieux et les dates de représentation.
Fonctionnalités Principales

Inscription et Authentification : Permet aux utilisateurs de créer un compte et de se connecter de manière sécurisée.
Consultation des Spectacles : Les utilisateurs peuvent consulter la liste des spectacles disponibles et accéder à leurs fiches détaillées.
Réservation de Places : Les utilisateurs peuvent réserver des places pour les spectacles de leur choix.
Gestion des Commentaires : Les utilisateurs ont la possibilité de laisser des commentaires et des avis sur les spectacles.

## Technologies et Outils Utilisés

# Backend :
- Langage : Java
- Framework : Spring Boot
- Environnement de développement : IntelliJ IDEA

# Base de Données :
- Système de gestion de base de données : MySQL via WampServer

# Frontend :
- Langages : HTML, CSS
- Framework CSS : Tailwind CSS
- Langage de script : JavaScript
